# Экспорт данных в Excel

Добавлена функциональность экспорта данных в Excel файлы на страницах админ-панели.

## Установка

Библиотека `xlsx` уже установлена в проекте:

```bash
pnpm add xlsx
```

## Страница Аналитики (/admin/analytics)

### Кнопка экспорта
Расположена в правом верхнем углу страницы, рядом с заголовком.

### Структура Excel файла

Файл экспортируется с названием `analytics_YYYY-MM-DD.xlsx` и содержит 3 листа:

#### Лист 1: "Сводка"
Содержит все метрики аналитики:
- **Онбординг**: первая вещь, прогресс гардероба (30%, 50%, 100%)
- **Aha-момент**: первый образ, первая примерка, клики по рекомендациям
- **Доставка ценности**: сохранённые образы, repeat task rate, образов на пользователя
- **Вовлечённость**: использование AI, количество AI сессий
- **Retention**: D1/D7/D30 метрики и количество вернувшихся пользователей
- **Монетизация**: paywall показан, конверсии, premium пользователи

#### Лист 2: "Воронка"
Таблица с данными воронки конверсии:
- Этап
- Количество пользователей

#### Лист 3: "Динамика"
Временной ряд за последние 30 дней:
- Дата
- Первая вещь (события)
- Первый образ (события)
- Образ сохранён (события)
- AI использован (события)

### Код экспорта

```typescript
const exportToExcel = () => {
  if (!data) return

  const wb = XLSX.utils.book_new()

  // Создание листов с данными
  const summaryData = [["Метрика", "Значение"], ...]
  const ws1 = XLSX.utils.aoa_to_sheet(summaryData)
  XLSX.utils.book_append_sheet(wb, ws1, "Сводка")

  // Экспорт
  const date = new Date().toISOString().split("T")[0]
  XLSX.writeFile(wb, `analytics_${date}.xlsx`)
}
```

## Страница Пользователей (/admin/users)

### Кнопка экспорта
Расположена в правом верхнем углу страницы, рядом с заголовком.

### Структура Excel файла

Файл экспортируется с названием `users_activity_YYYY-MM-DD.xlsx` и содержит 4 листа:

#### Лист 1: "Пользователи"
Таблица со всеми пользователями и их активностью:
- ID (первые 8 символов user_id)
- Имя
- Email
- Статус (Админ / Pro / Free)
- Кредиты
- AI запросов (лимит)
- Вещей в гардеробе (лимит)
- Дата регистрации
- Последнее обновление

#### Лист 2: "Метрики"
Сводные метрики:
- Всего пользователей
- MAU (Monthly Active Users)
- DAU (Daily Active Users)
- Активных подписок

#### Лист 3: "Активность"
График активности за 30 дней:
- Дата
- Активных пользователей

#### Лист 4: "Регистрации"
График регистраций за 30 дней:
- Дата
- Количество регистраций

### Код экспорта

```typescript
const exportUsersToExcel = () => {
  if (users.length === 0) return

  const wb = XLSX.utils.book_new()

  // Создание листа с пользователями
  const userData = [
    ["ID", "Имя", "Email", "Статус", "Кредиты", ...],
    ...users.map((user) => [...])
  ]
  const ws1 = XLSX.utils.aoa_to_sheet(userData)
  XLSX.utils.book_append_sheet(wb, ws1, "Пользователи")

  // Экспорт
  const date = new Date().toISOString().split("T")[0]
  XLSX.writeFile(wb, `users_activity_${date}.xlsx`)
}
```

## Использование

### На странице аналитики:
1. Перейдите на `/admin/analytics`
2. Нажмите кнопку "Экспорт в Excel" в правом верхнем углу
3. Файл `analytics_YYYY-MM-DD.xlsx` будет загружен автоматически

### На странице пользователей:
1. Перейдите на `/admin/users`
2. Нажмите кнопку "Экспорт в Excel" в правом верхнем углу
3. Файл `users_activity_YYYY-MM-DD.xlsx` будет загружен автоматически

## Технические детали

### Библиотека xlsx
- **Версия**: 0.18.5
- **Документация**: https://docs.sheetjs.com/
- **GitHub**: https://github.com/SheetJS/sheetjs

### Методы XLSX

```typescript
// Создание новой книги
const wb = XLSX.utils.book_new()

// Создание листа из массива массивов
const ws = XLSX.utils.aoa_to_sheet(data)

// Добавление листа в книгу
XLSX.utils.book_append_sheet(wb, ws, "Название листа")

// Сохранение файла
XLSX.writeFile(wb, "filename.xlsx")
```

### Формат данных

Данные передаются как массив массивов (Array of Arrays):

```typescript
const data = [
  ["Заголовок 1", "Заголовок 2", "Заголовок 3"],  // Первая строка - заголовки
  ["Значение 1", "Значение 2", "Значение 3"],      // Данные
  ["Значение 4", "Значение 5", "Значение 6"],
]
```

## Преимущества

1. **Простота**: Одна кнопка для скачивания всех данных
2. **Структурированность**: Данные разделены по листам для удобства анализа
3. **Совместимость**: Excel файлы открываются в Excel, Google Sheets, LibreOffice
4. **Дата в названии**: Легко отслеживать версии экспортов
5. **Автоматическое форматирование**: Библиотека сама определяет типы данных
